# 🛍️ Customer Shopping Behavior Analytics

An end-to-end **Data Analytics** project that transforms raw customer shopping data into meaningful business insights through data cleaning, SQL analysis, interactive dashboards, and modern data visualization.

The project demonstrates the complete analytics workflow—from raw CSV data to business-ready dashboards using **Python, SQL Server, Power BI, and a custom Web Dashboard**.

---

## 📌 Project Overview

This project analyzes customer shopping behavior to uncover purchasing patterns, customer demographics, subscription trends, product performance, seasonal sales, and revenue distribution.

The project follows a complete Business Intelligence pipeline:

CSV Dataset → Python → Data Cleaning → SQL Server → SQL Analysis → Power BI Dashboard → Web Dashboard → Report → Presentation

---

The dataset contains approximately **3,900 customer records** with information including:

## 📂 Project Workflow

### 1. Data Loading
- Imported the customer shopping dataset into Python using Pandas.

### 2. Exploratory Data Analysis (EDA)
- Examined dataset structure
- Generated descriptive statistics
- Identified missing values
- Explored customer purchasing patterns

### 3. Data Cleaning & Preprocessing
- Handled missing values
- Standardized column names
- Performed feature engineering
- Removed redundant columns
- Validated data consistency

### 4. Database Integration
- Connected Python with Microsoft SQL Server.
- Loaded the cleaned dataset into SQL Server.
- Created database tables for analysis.

### 5. SQL Business Analysis
Executed SQL queries to answer business questions, including:

- Revenue by Gender
- Revenue by Age Group
- Subscription Analysis
- Customer Segmentation
- Product Performance
- Shipping Analysis
- Discount Analysis
- Top Rated Products
- Repeat Buyer Analysis

## 📊 Key Business Insights

- Clothing generated the highest revenue among all product categories.
- Customers aged **25–34** contributed the largest share of revenue.
- Male customers generated higher total revenue than female customers.
- Loyal customers represented the largest customer segment.
- Nearly half of all purchases involved discounts.
- Non-subscribers contributed the majority of total revenue.
- Seasonal trends revealed variations in purchasing behavior throughout the year.

## 🚀 How to Run

### Python Analysis
1. Open the Jupyter Notebook.
2. Install the required Python libraries.
3. Execute all notebook cells.

### SQL Server
1. Create the SQL Server database.
2. Import the cleaned dataset.
3. Run the SQL scripts.

### Power BI
1. Open the `.pbix` file.
2. Connect it to the SQL Server database.
3. Refresh the data.

### Web Dashboard
1. Open the `Web Dashboard` folder.
2. Launch `index.html` in a web browser.
3. Or visit the deployed dashboard using the live demo link.
